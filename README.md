<h3 align="left">My Projects on AI, LLM, and Computer Vision</h3>

- **Smart Detection System for Emotional Patterns & Mood Swings | Computer Vision + LLM**  
  - Built a real-time emotion and identity recognition system using InsightFace & DeepFace libraries, integrated with Claude 3 Haiku chatbot to generate conversational insights on individual emotional trends and patterns.  
  - Designed for deployment in mental health facilities, nursing homes, interrogation rooms, and schools to support behavioral monitoring and intervention.

- **Legal Chatbot Assistant for Small Business Owners | AI + RAG**  
  - Developed a legal Q&A assistant using Claude 3 Haiku and Vector RAG, providing instant answers on intellectual property laws and related policies.  
  - Streamlined access to legal information for entrepreneurs, reducing research time and consultation costs.

- **Automated ID Matching System for Large-Scale Event Entrances | Computer Vision + Anti-Spoofing**  
  - Implemented InsightFace-based ID-to-face matching with live webcam verification for large events (e.g., international conferences).  
  - Integrated ONNX-based anti-spoofing model, reducing entrance processing time and manpower needs for event organizers.
